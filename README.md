## 📖 About

**SemiVerse** is a modern educational platform designed to simplify semiconductor concepts using interactive visualizations and animations. Instead of static textbook diagrams, users can explore electron movement, PN junctions, energy bands, and semiconductor devices in an engaging way.

Whether you're an engineering student, educator, or electronics enthusiast, SemiVerse makes learning semiconductor physics intuitive and enjoyable.

Interactive and modern UI

Conductors, Insulators & Semiconductors

Beginner-friendly explanations

Run
